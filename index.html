📚 O que é um Cookie?

Um cookie é um pequeno arquivo de texto que o navegador armazena no computador do usuário. É usado para:

  • Manter dados de login - Lembrar o usuário logado
  • Preferências do usuário - Tema escuro, idioma, etc.
  • Rastrear comportamento - Análise de uso (Google Analytics)
  • Carrinho de compras - Manter produtos no carrinho
  • Publicidade - Mostrar anúncios relevantes

Estrutura de um Cookie:

document.cookie = "nome=valor; expires=data; path=/; domain=.exemplo.com"
  • nome=valor - Dados do cookie
  • expires - Quando o cookie expira
  • path - Em quais páginas o cookie é acessível
  • domain - Em quais domínios o cookie é acessível
  • secure - Apenas em conexões HTTPS
  • samesite - Proteção contra CSRF
